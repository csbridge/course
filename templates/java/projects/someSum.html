Problem written by Bryce C-R, Julia Lee, Chris Piech, and TY Huang

Day2: SomeSum.java

Write a program that reads in 10 integers from the user and then outputs the sum of the entered values.

Often in console programs it is necessary to print out values to the user in a string. The example above is:

"The sum is: 55"

If you run your program with the above input, 55 will be the value stored in the variable that tracks the sum. If that variable is named exampleSum, then in order to output its value you would write:

println("The sum is: " + exampleSum);

A more general form is:

"String message" + variableName + "more string message"
