This is the first lab problem for the day. It gives students practice writing their first Karel program. We want to emphasize decomposition and defining their own Karel commands.

Newspaper Karel Solution

public class CollectNewspaper extends Karel {

    public void run() {
        moveToNewspaper();
        pickUpNewspaper();
        returnToHouse();
    }

    private void moveToNewspaper() {
        move();
        move();
        turnRight();
        move();
        turnLeft();
        move();
    }

    private void pickUpNewspaper() {
        pickBeeper();
    }

    private void returnToHouse() {
        turnAround();
        move();
        move();
        move();
        turnRight();
        move();
        turnRight();
    }

    private void turnAround() {
        turnLeft();
        turnLeft();
    }

    private void turnRight() {
        turnLeft();
        turnLeft();
        turnLeft();
    }

}
